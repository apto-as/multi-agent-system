# TMWS の4つの核心特徴

TMWSは Trinitas-agents（9つのエージェント）に以下の4つのファクターを与えるシステムです：

## 1. 記憶 (Memory) - 第一の特徴
- ChromaDB によるセマンティックベクトル検索
- エージェント間での知識共有
- 永続的な学習履歴の保存

## 2. ナラティブ (Narrative) - 第二の特徴
- エージェントの文脈理解
- ストーリーテリングによる情報伝達
- コンテキストの継続性

## 3. スキル (Skills) - 第三の特徴
- 動的スキル作成・共有・実行
- MCPツールとしての活性化
- エージェント間でのスキル共有

## 4. 学習 (Learning) - 第四の特徴
- パターン認識と適応
- 信頼スコアシステム
- 継続的な改善

## 重要性
これらの4つの特徴が Trinitas エージェントに知性と継続性を与え、単なるツール実行を超えた協調的な問題解決を可能にします。

## 設計原則
Tool Search Tool や MCP ハブの実装において、これらの4つの特徴を損なわないことが必須条件です。
